# GEM_representative_insicision

Unified ultra-rank-frequency controller for generating representative profiles from geophysical (EC/MS) line measurements.

This repository contains `RIs_v2.py`, a script that is the basis of a **Streamlit application** that acts similarly to `RIs_v1.py` script while offering a UI and more choices regarding interpolation.

Interpolation methods included:
1. Linear
2. Cubic
3. Nearest
4. Quadratic
